## Description

This is a curated reference of notable temporary email services as of 2025. Services were chosen for reliability, speed, feature set (inbox persistence, domain options, API access), privacy, and overall usefulness.

## Services (curated)

1. **Temp-Mail** - Popular, wide domain selection, auto-refresh inbox, browser extension.

   - Features: Multiple domains · Auto-refresh inbox · Browser extension
   - Link: https://temp-mail.org

2. **Mail.tm** - Privacy-focused, API access, inbox persistence.

   - Features: Privacy-focused · API · Inbox persistence
   - Link: https://mail.tm
   - Recommended: **Yes** - Recommended for automation, privacy-conscious use, and developers who need API access.

3. **Guerrilla Mail** - One of the oldest disposable-mail services, supports address customization and multiple domains.

   - Features: Address customization · Multiple domains
   - Link: https://www.guerrillamail.com

4. **10MinuteMail** - Extremely simple; inbox expires in 10 minutes (extendable).

   - Features: Very ephemeral · Extendable timer
   - Link: https://10minutemail.com

5. **EmailOnDeck** - Fast, no-signup required, friendly for quick signups.
   - Features: Fast · No sign-up
   - Link: https://emailondeck.com

## Privacy Scorecards

Each service below includes a short, opinionated privacy & features scorecard. Scores are subjective and based on public info as of 2025.

- Logging: Low / Medium / High - how likely the service logs identifying request data.
- Retention: Short / Medium / Long - how long messages are typically stored.
- TLS: Yes / No - if the site uses HTTPS for transport.
- API: Yes / No - if a documented API exists for automation.

Temp-Mail

- Logging: Medium
- Retention: Medium
- TLS: Yes
- API: Partial / Unofficial

Mail.tm

- Logging: Low
- Retention: Medium (addresses can persist longer)
- TLS: Yes

10MinuteMail
